# CodeFlow AI

## Project Overview
CodeFlow AI is an interactive educational tool that helps beginners understand how Python programs execute internally.

### Features
- Line-by-line execution tracing
- Memory visualization
- State table generation
- Syntax error detection
- Runtime error explanation
- Play/Pause execution controls

### Tech Stack
- Python
- Streamlit
- AST Module
- sys.settrace()

### Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
